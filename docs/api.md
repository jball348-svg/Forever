## debounce.js

*Debounce and throttle utilities.*

### `debounce`

Delay invoking `fn` until `delayMs` ms have elapsed since the last call.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `fn` | `Function` | Function to debounce |
| `delayMs` | `number` | Delay in milliseconds |

**Returns:** `{Function}` Debounced wrapper

**Example:**
```js
const save = debounce(() => api.save(), 300);
window.addEventListener('input', save);
```

---

## retry.js

*Retry an async function with configurable attempts and delay.*

### `retry`

Retries an async function up to `options.attempts` times with a delay between each attempt.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `fn` | `Function` | Async function to call |
| `options` | `{ attempts?: number, delayMs?: number }` | Retry configuration |

**Returns:** `{Promise<*>}` Resolves with the function's return value

**Example:**
