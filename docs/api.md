### `debounce`

Delay invoking `fn` until `delayMs` milliseconds have elapsed since the last call. Useful for handling rapid user input events like typing or resizing.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `fn` | `Function` | - The function to debounce |

**Example:**
```js

```

### `throttle`

Invoke `fn` at most once per `intervalMs` milliseconds. Calls that occur within the interval are silently dropped.

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `fn` | `Function` | - The function to throttle |
| `intervalMs` | `number` | - Minimum interval between invocations in milliseconds |

**Returns:** `{Function} A throttled wrapper around `fn``

**Example:**
