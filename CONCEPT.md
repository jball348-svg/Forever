# Forever — Concept Document

[![CI](https://github.com/jball348-svg/Forever/actions/workflows/ci.yml/badge.svg)](https://github.com/jball348-svg/Forever/actions/workflows/ci.yml)

## What is this?

**Forever** is a self-improving repository with a simple but compelling premise: it evolves one task at a time, forever.

At any given moment, the repo contains a single pending task in a file called `next_task`. That task is completed — code written, files created, structure improved — and then replaced with a new task that pushes the project one step further. Rinse, repeat, indefinitely.

The agent doing the work could be a human, an AI, or both. The repo doesn't care. It just keeps going.

---

## What does it aim to become?

That's the open question — and that's the point.

Forever isn't building toward a fixed goal. It's an experiment in emergent direction. Each iteration shapes what the next iteration could be. Over time, a project identity should naturally crystallise from the accumulation of decisions, files, and improvements.

It might become a tool. It might become a game. It might become a piece of conceptual art. It will become *something* — we just don't decide that up front.

---

## The Rules

These are the only rules, and they live in the README:

1. **Read `next_task`.** Do whatever it says, completely and with care.
2. **Replace `next_task`** with a new task that meaningfully improves the project. Use your judgement.
3. **Never edit the README.** It is the constitution. It does not change.
4. **Never do anything to `next_task` other than replace its content.** No deleting, no renaming, no archiving.

That's it. Everything else is fair game.

---

## Why join in?

Because it's genuinely interesting to watch something grow without a predetermined destination.

Because the constraint is liberating — you only ever have to do one thing.

And because somewhere down the line, when this repo has hundreds of commits and a real identity, you'll be able to trace it all back to: *give this project a working title.*

Welcome to **Forever**.
